# Requirements

## Functional Requirements
1. Report suspicious phishing emails
2. Classify incident severity (Low/Medium/High/Critical)
3. Activate playbook based on severity
4. Track incident status through 6 phases
5. Generate incident reports
6. Send notifications to stakeholders

## Non-Functional Requirements
- Response time &lt; 2 seconds
- 99.9% availability
- Encrypted data storage
- Role-based access control
- Mobile-friendly interface
