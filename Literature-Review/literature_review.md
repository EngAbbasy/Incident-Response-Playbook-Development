
# 📚 Literature Review

## 📌 Introduction
Phishing attacks are among the most common cybersecurity threats, exploiting human behavior through social engineering techniques to steal sensitive information such as credentials and financial data.

---

## 📖 Review of Existing Work

### 🔹 NIST SP 800-61 (Computer Security Incident Handling Guide)
This framework defines the standard incident response lifecycle:
- Preparation  
- Identification  
- Containment  
- Eradication  
- Recovery  
- Lessons Learned  

It serves as the foundation for structuring the incident response playbook.

---

### 🔹 MITRE ATT&CK Framework (T1566 – Phishing)
MITRE ATT&CK provides detailed techniques used in phishing attacks, including:
- Spear Phishing Emails  
- Malicious Links  
- Credential Harvesting  

This framework was used to analyze attacker behavior and design detection and response strategies.

---

### 🔹 SANS Incident Handling Guide
The SANS guide emphasizes:
- Rapid detection and response  
- Minimizing impact  
- Structured incident handling procedures  

It influenced the practical implementation of the playbook.

---

### 🔹 Related Research Papers
Several studies focus on phishing detection techniques, including:
- Machine learning-based phishing detection  
- Email filtering and classification systems  

However, most research focuses on detection rather than full incident response.

---

## 📊 Gap Analysis
Most existing studies focus only on detecting phishing attacks.  
There is limited focus on full incident response handling.

This project addresses the gap by covering:
- Detection  
- Response  
- Recovery
